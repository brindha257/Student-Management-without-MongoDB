A web-based application for managing student data, built with Node.js and Express.js. This portal allows administrators to view, add, and delete student records through a user-friendly interface.

Features
View a list of all students with details such as Roll No, Name, Age, Grade, Feedback, Score, and Passed Year

Add new students via a simple form

Delete existing student records

Responsive and clean user interface using EJS templates

Tech Stack
Backend: Node.js, Express.js

Frontend: HTML, CSS, EJS (Embedded JavaScript templates)

Dependencies: body-parser, ejs, express
